# Chat App (MERN + Socket.io)

This is a real-time chat application built using the MERN stack. It allows users to sign up, log in, chat in real time, and send images. The app also shows online users and lets users update their profile picture.

---

## 🚀 What this project does

- Users can create an account and log in securely  
- Real-time messaging using Socket.io  
- Send text messages and images  
- See online users instantly  
- Update profile picture (stored in Cloudinary)  
- Responsive UI for mobile and desktop  

---

## 🧰 Tech Stack

### Frontend
- React (Vite)
- Zustand (state management)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.io
- Cloudinary

---

## 📁 Project Structure

- **Frontend** → UI components, pages, and state management stores  
- **Backend** → APIs, authentication, database models, and Socket.io real-time communication
