Chat App (MERN + Socket.io)

This is a real-time chat application built using the MERN stack. It allows users to sign up, log in, chat in real time, and send images. The app also shows online users and lets users update their profile picture.

🚀 What this project does
1.Users can create an account and log in securely
2.Real-time messaging using Socket.io
3.Send text messages and images
4.See online users instantly
5.Update profile picture (stored in Cloudinary)
6.Responsive UI for mobile and desktop

🧰 Tech used
Frontend
React (Vite)
Zustand for state management
Tailwind CSS
Axios

Backend
Node.js
Express.js
MongoDB (Mongoose)
Socket.io
Cloudinary

📁 Project structure
Frontend contains UI components, pages, and stores for state management.
Backend handles APIs, authentication, database models, and real-time socket communication.
